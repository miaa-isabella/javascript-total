evento on click, —Este atributo responde al evento click. El evento se ejecuta cuando el usuario hace clic con el botón izquierdo del ratón. HTML ofrece otros dos atributos similares llamados ondblclick (el usuario hace doble clic con el botón izquierdo del ratón) y oncontextmenu (el usuario hace clic con el botón derecho del ratón).



evento onmousedow, Este atributo responde al evento mousedown. Este evento se desencadena cuando el usuario pulsa el botón izquierdo o el botón derecho del ratón.



evento onmouseenter, onmouseenter—Este atributo responde al evento mouseenter. Este evento se desencadena cuando el ratón se introduce en el área ocupada por el elemento.



evento onmouseleave, —Este atributo responde al evento mouseleave. Este evento se desencadena cuando el ratón abandona el área ocupada por el elemento.



evento onmouseout, Este atributo responde al evento mouseout. El evento se desencadena cuando el ratón abandona el área ocupada por el elemento o cualquiera de sus elementos hijos.



evento onmousemove, —Este atributo responde al evento mousemove. Este evento se desencadena cada vez que el ratón se encuentra sobre el elemento y se mueve.



evento onwheel, Este atributo responde al evento wheel. Este evento se desencadena cada vez que se hace girar la rueda del ratón.



evento onkeypress, —Este atributo responde al evento keypress. Este evento se desencadena cuando se activa el elemento y se pulsa una tecla.
